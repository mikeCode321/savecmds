# savecmds

Save and reuse terminal commands directly from the VS Code sidebar.

## Features

- **Save commands** — store any terminal command with a custom name
- **Run commands** — click a saved command to instantly run it in the terminal
- **Folders** — organize commands into folders
- **Drag and drop** — move commands between folders
- **Edit** — right click any command or folder to rename or edit
- **Delete** — right click any command or folder to delete it
- **Persistent** — commands are saved across VS Code sessions

## How to Use

1. Click the terminal icon in the Activity Bar on the left
2. Click the **+** button to save a new command
3. Click the **folder** button to create a folder
4. Click any saved command to run it in the terminal
5. Right click any command or folder for more options
